# Interactive Spotify API Dashboard
This dashboard allows users to explore and analyze music data from the Spotify API. Users can visualize artist statistics, track features, album information, and generate insights about their favorite songs.

## Technologies Used
- Streamlit for creating the interactive web application
- Spotify API for accessing music data
- Pandas for data manipulation
- Matplotlib and Seaborn for data visualization

## Features
- Explore detailed analytics about artists and tracks
- Visualize song statistics such as popularity, danceability, and energy
- Search for artists and their top tracks

## Data Source
The data used in this dashboard is sourced from the Spotify API, providing real-time access to a vast music database.
